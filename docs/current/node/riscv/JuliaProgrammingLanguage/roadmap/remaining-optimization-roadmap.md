# 🟣 Remaining Optimization Roadmap

**Last Updated:** 2026-03-11

## Candidate Themes

- Integer and modular arithmetic kernels.
- Vector-friendly residue or sieve strategies.
- Memory-layout experiments for cache-aware traversal.
- Serialization helpers for reproducible binary fixtures.

## Review Gates

Each optimization idea should be tracked with:

1. a Julia reference implementation,
2. a parity fixture against C++, and
3. a benchmark note that describes hardware and inputs.
