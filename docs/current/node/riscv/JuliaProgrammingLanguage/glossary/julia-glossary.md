# 🟣 Julia Glossary

**Last Updated:** 2026-03-11

| Term | Meaning in this section |
|---|---|
| Julia project | A local Julia environment with reproducible package resolution |
| kernel | A focused math or vector routine under experiment |
| prototype | A non-production Julia implementation used to validate ideas |
| parity fixture | A shared input/output case that must match C++ behavior |
| PackageCompiler | Julia tooling for producing local app or library artifacts |
