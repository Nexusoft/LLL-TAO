# 🟣 Julia/C++ Interop Dictionary

**Last Updated:** 2026-03-11

| Term | Meaning |
|---|---|
| SIM-LINK | Documentation shorthand for a defined Julia ↔ C++ interface boundary |
| embedding | Hosting the Julia runtime inside a C++ process |
| `ccall` | Julia mechanism for calling C-compatible library symbols |
| shared library | A compiled boundary that Julia or C++ can load explicitly |
| offline workflow | Running Julia outside the production node to validate an idea before porting |
