# 🟣 Crypto Scope and Boundaries

**Last Updated:** 2026-03-11

Julia-based crypto or number-theory experiments must stay clearly separated from production cryptographic paths until reviewed. Experimental code may help reason about algorithms, but production safety depends on the existing C++ and library-backed implementations.
