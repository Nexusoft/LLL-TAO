◆ GetUnifiedAverage()

int GetUnifiedAverage ( )

GetUnifiedAverage

Gets the Current Unified Time Average. The More Data in Time Average, the less a pesky node with a manipulated time seed can influence. Keep in mind too that the Unified Time will only be updated upon your outgoing connection... otherwise anyone flooding network with false time seed will just be ignored. The average is a moving one, so that iter_swap will allow clocks that operate under different intervals to always stay synchronized with the network.
